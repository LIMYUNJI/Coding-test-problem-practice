📡 1. 네트워크 기본 개념
네트워크(Network): 컴퓨터/장치들이 데이터를 주고받는 구조
프로토콜(Protocol): 통신 규칙 (형식, 순서, 오류 처리 등)

👉 대표 프로토콜

HTTP, FTP, SMTP, TCP, IP 등
🌐 2. OSI 7계층 (⭐ 핵심)

시험에서 가장 중요

계층	역할	주요 프로토콜/장비
7 응용	사용자 인터페이스	HTTP, FTP
6 표현	암호화, 압축	JPEG, MPEG
5 세션	연결 관리	API
4 전송	신뢰성, 흐름제어	TCP, UDP
3 네트워크	경로 설정	IP, 라우터
2 데이터링크	오류검출	MAC, 스위치
1 물리	전송	케이블

👉 외우는 팁
응표세전네데물

🌍 3. TCP/IP 4계층

실제 인터넷 구조

계층	설명
응용	HTTP, FTP
전송	TCP, UDP
인터넷	IP
네트워크 인터페이스	LAN
🚚 4. TCP vs UDP (⭐ 자주 출제)
구분	TCP	UDP
연결	O	X
신뢰성	높음	낮음
속도	느림	빠름
특징	순서보장	실시간

👉 정리

TCP → 정확 (파일, 웹)
UDP → 빠름 (영상, 게임)
📍 5. IP 주소
IPv4: 32bit (예: 192.168.0.1)
IPv6: 128bit (길고 복잡)

👉 종류

공인 IP / 사설 IP
클래스 A/B/C (시험 자주 나옴)
🧮 6. 서브넷 (Subnet)
IP를 네트워크/호스트로 나눔
서브넷 마스크 사용

👉 예

255.255.255.0 → 앞은 네트워크, 뒤는 호스트
