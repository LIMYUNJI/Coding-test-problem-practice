소프트웨어 설계
현행 시슽템 분석에는 플랫폼 분석 기능, 플랫폼 성능 특성 분석, 운영 체게 분석, DBMS분석, 운영체제 분석, 비즈니스 융합 분석이 있다.

멀티 버스 활성화를 위한 CPND 의미
C : 콘텐츠 P : 플랫폼 N : 네트워크 D : 디바이스

플랫폼이란 수많은 애플리케이션을 실행하는데 상호호환이 쉽게 이루어질 수 있도록 만들어진 환경을 의미한다

플랫폼의 기능
1. 연결기능 2. 비용감소 기능 3. 검색 비용 절약 기능 4. 커뮤니티 형성에 의한 네트워크 효과

운영체제 
하드웨어와 소프트 웨어 자원을 관리하고 컴퓨터의 공통 서비스를 제공하는 소프트웨어를 말한다.

운영체제의 종류
1. 윈도우 - 일반 ㅖC 유지등 관리 비용 장점
2. 유닉스 - 대용량 처리, 안전성 높은 엩터프라이즈 급 서버
3. 리눅스 - 중/대규모 서버 높은 보안성 제공
4. IOS - 다양한 기기와의 홓 ㅘㄴ성 제공
5. 안드로이드 - 높은 보안성과 고성능 제공
